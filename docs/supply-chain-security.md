## SLSA Alignment

This project aligns with **SLSA Level 3** expectations:

- ✔️ Version-controlled, reviewed source
- ✔️ Isolated, ephemeral build runners
- ✔️ Non-interactive, scripted builds
- ✔️ Artifact signing with OIDC (Cosign)
- ✔️ SBOM generation and attestation
- ✔️ Build provenance attestation
- ✔️ Protected release tags and workflows
